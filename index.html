Summary Of Weather Plots
Latitude in comparison to other weather Variables

For this homework we'll be creating a visualization dashboard website using visualizations we've created in a past assignment. Specifically, we'll be plotting weather data for 547 cities accross the globe. The source of the data was through an API call to OpenWeatherMap API


The main Python Library that was utilized to the model the data was Matplot Library and this was used to form a correlation between Latitude (distance from the equator) and several other variables such as Wind Speed, Cloudiness, Humidity and Temparautre
